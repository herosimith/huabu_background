```json

  "model": "gpt-image-2",
  "size": "1024x1024",
  "quality": "high",

  "background": "transparent"
}
```

## Positive prompt

生成一张用于深色 AI 广告生产工作台的单角色 loading 人物资产，1:1 方形画布，透明背景。

主体是一位性别中性的年轻成年广告创意操作员，轻度卡通化但保持专业感，约 1:3.5 头身比例，不要幼儿化。短深色头发，专注而平静的微笑，穿石墨灰色现代工作夹克，衣领带一小段清晰的亮蓝色饰边。人物以四分之三侧面坐姿微微前倾，左手扶住深灰色数位绘图板，右手握细长触控笔，正在绘制一条青绿色发光线稿。人物周围只保留三个小型方形色板，分别为亮蓝、薄荷绿和暖金色，用来暗示广告配色正在生成。

构图居中，人物与绘图板占画面约 68%，四周保留至少 14% 透明安全边距，头发、手臂、触控笔和绘图板全部完整，不得裁切。角色在 220-280px 的 UI 展示尺寸下仍能清楚辨认轮廓、面部表情、握笔动作和绘图板。

视觉风格为克制的 2.5D editorial cartoon，哑光软胶与轻微黏土质感，边缘干净，造型简练，阴影短而柔和，不要写实皮肤。配色严格贴合专业深色 SaaS 工作台：石墨黑 #101318、深灰蓝 #293642、亮蓝 #4AA8FF、薄荷绿 #34D399、少量暖金 #F6C56D，肤色自然偏暖。整体安静、可靠、有创作专注感，不要活泼营销感，不要赛博朋克，不要紫色霓虹。

仅生成一个完整角色和必要的绘图板、触控笔、三个色板。图片中不出现任何文字、字母、数字、按钮、界面面板、品牌标识、Logo 或水印。单张最终成品，不要角色设定表、过程图、网格拼贴、样机展示板或多姿势版本。输出边缘适合透明 PNG 叠加在 #0B0F14 深色背景上。
